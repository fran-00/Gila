About Gila

What is Gila?

Gila is a client that provides a graphical interface to connect to generative AI language models.

How to use Gila?

API Keys

In order to use the program, you need to obtain at least an API key: this is a unique code that is assigned to an application or web service to allow access and interaction with the provider. Once you have entered an API key into the program, you can start a new conversation with one of the available language models. Using some services involves costs.

The toolbar

From the toolbar you can perform these actions:

  1. Export the current conversation as a .txt or .docx file.
  2. Add API keys to communicate with services.
  3. Change conversation settings such as model, temperature, and maximum tokens that can be used in a reply.
  4. Open this window with program information.

Saved chats

Conversations are automatically saved in the sidebar: the name assigned is a simple 10-letter ID that serves to uniquely identify the conversation, but it is possible to rename them to distinguish them more easily and delete them.

Temperature

Temperature is a value that represents the model's degree of creativity: a higher temperature means more creative and unpredictable responses. You can change the temperature value from the settings window.

Tokens

Tokens are minimal units of character sequences in a text. Large language models (like GPT) analyze a text by dividing it into tokens and starting from the statistical relationships that exist between the character sequences they are able to produce a coherent text in response. In English a token is equivalent to about 4 characters.

The program will show the number of tokens used for a given conversation: how many tokens constitute the question, how many tokens constitute the answer and the total number of tokens.

Issues

It may happen that the program crashes, especially due to connection problems with the web services needed for it to work: in this case, just close Gila and restart it.
