Informazioni su Gila

Che cos'è Gila?

Gila è un client che fornisce un'interfaccia grafica per connettersi al modelli linguistici di intelligenza artificiale generativa, creato nello specifico per aiutare ad espandere racconti.
Al momento supporta OpenAI, Cohere e supporterà Google API quando l'utilizzo del servizio sarà disponibile anche in Italia.

Come si utilizza Gila?

Per poter utilizzare il programma bisogna ottenere le chiavi API per avere accesso ai servizi. Ogni servizio ha un sistema di pagamento diverso, ma sono tutti basati sul numero di token utilizzati. Al momento Cohere Chat è gratis mentre i modelli di OpenAI sono a pagamento.

La barra degli strumenti

Dalla barra degli strumenti è possibile compiere 4 azioni:

  1. Esportare la conversazione corrente come file .txt o .docx.
  2. Aggiungere le chiavi API per poter comunicare con OpenAI e Cohere.
  3. Modificare le impostazioni relative alla conversazione come il modello, la temperatura e il numero di token massimi utilizzabili in una risposta.
  4. Aprire questa finestra con le informazioni sul programma.

Le chat salvate

Le conversazioni vengono salvate automaticamente nella barra laterale: il nome assegnato è un semplice ID a 10 caratteri che serve per identificare univocamente la conversazione, ma è possibile rinominarle per distinguerle più facilemente e cancellarle.

La temperatura

La temperatura è un valore che rappresenta il grado di creatività del modello: a una temperatura più alta corrispondono risposte più fantasiose e imprevedibili. Si può cambiare il valore della temperatura dalla finestra delle impostazioni.

I token

I token sono unità minime di sequenze di caratteri di un testo. I grandi modelli linguistici (come GPT) analizzano un testo dividendolo in token e a partire dalle relazioni statistiche che ci sono tra le sequenze di caratteri sono in grado di produrre del testo coerente.
In lingua inglese un token equivale a circa 4 caratteri e 1,000 token corrispondono a circa 750 parole.
Il programma mostrerà il numero di token utilizzati per una conversazione data: quanti token costituiscono la domanda, quanti token costituiscono la risposta e il numero di token complessivo.
