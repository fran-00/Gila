About Gila

What is Gila?

Gila is a client that provides a graphical interface to connect to generative AI language models.

How to use Gila?

API Keys

To use Gila, you need an API key, a unique code that grants access to AI providers. Once entered, you can start a conversation with an available model. Some services may involve costs.

Tokens

Tokens are the smallest text units AI models process. A model analyzes input by splitting it into tokens and generates output based on learned patterns. In English, one token is about four characters.

Gila estimates token usage before sending a message, but the count varies by model. The exact token usage is updated in real time after each response.

The Sidebar

The sidebar contains two tabs: Saved Chats and Settings.

Saved Chats

Conversations are automatically saved and assigned a unique 10-character ID, which can be renamed or deleted.

Settings

You can configure model selection, temperature, max tokens, and system messages. These settings apply to new conversations.

  • Model - Select a language model from the dropdown. If an API key is missing for a model, Gila will prompt you to enter one.
  • Temperature - Controls response randomness: higher values yield more creative but unpredictable outputs.
  • Max Tokens - Sets the upper limit of generated tokens. Models may stop earlier based on their internal logic.
  • System Message - A predefined instruction guiding AI behavior and context, useful for defining tone, rules, or specific tasks.

Advanced reasoning models (e.g., OpenAI’s o-series) use different parameters:

  • Reasoning effort - Determines how many reasoning tokens the model uses before generating a response. Higher values improve problem-solving but increase latency and cost.
  • Max Completion Tokens - Limits the combined total of reasoning and response tokens.

For image generation, other parameters are available:

  • Size - Defines image dimensions, which vary by model.
  • Quality - The quality of the image you want to generate. Higher quality (HD) is available with DALL-E 3.
  • N. of images - Controls the number of variations (only for DALL-E 2).

Note: Image links generated by AI models expire over time. Save them immediately by right-clicking on the image.

While OpenAI allows generating up to 10 images with DALL-E 2, actual limits depend on your account’s rate restrictions.

The toolbar

The toolbar allows you to:

  1. Export conversations as .txt, .pdf, or .docx.
  2. Manage API keys.
  3. Access this help window.

Rate Limits

API providers impose rate limits on requests based on usage tier. Limits vary by model and service provider. Gila does not track these limits, but a popup will notify you if a request exceeds them.

Issues

Connection failures or service disruptions may cause crashes. Restart Gila if needed.
